} as const

**truncateSummary function:**
- Accepts `text: string`, returns `{ text: string; truncated: boolean }`
- If `text.length <= HEYGEN_CHAR_LIMIT`, return `{ text, truncated: false }`
- Otherwise, take `text.slice(0, HEYGEN_CHAR_LIMIT)`, search for last occurrence of `.` or `!` or `?` followed by a space or end-of-string using regex `/[.!?](?:\s|$)/` searching backward from the limit
- If sentence boundary found (position > 0), cut at that position + 1 (include the punctuation), trim trailing whitespace
- If no sentence boundary found, hard-cut at `HEYGEN_CHAR_LIMIT`
- Return `{ text: truncatedText, truncated: true }`

**chunkBeats function (not exported, internal helper):**
- Generic: `function chunkBeats<T>(items: T[], size: number): T[][]`
- Standard chunk loop: `for (let i = 0; i < items.length; i += size)` pushing `items.slice(i, i + size)`
- If `items.length === 0`, return `[[]]` (one empty chunk)

**beatToScene function (not exported, internal helper):**
- Accepts `beat: StoryBeat, opts: AdaptOptions`
- Returns `{ scene: HeyGenScene; warning: string | null }`
- Calls `truncateSummary(beat.summary)` for the voice input_text
- Builds `HeyGenScene` with:
  - `character.type: 'avatar'`, `character.avatar_id: opts.avatarId`, `character.avatar_style: opts.avatarStyle` (only include if defined)
  - `voice.type: 'text'`, `voice.input_text: truncated text`, `voice.voice_id: opts.voiceId`, `voice.speed: opts.speed` (only include if defined)
  - `background.type: 'color'`, `background.value: BEAT_COLOR_MAP[beat.type]`
- If truncated, set warning to `Beat "${beat.title}" summary truncated from ${original length} to ${truncated length} characters at sentence boundary`
- Return `{ scene, warning }`

**adaptStoryArc function (exported):**
- Signature: `export function adaptStoryArc(arc: StoryArc, opts: AdaptOptions): AdaptResult`
- Map `arc.beats` through `beatToScene`, collecting scenes and non-null warnings
- Chunk the scenes array with `chunkBeats(scenes, HEYGEN_MAX_SCENES)`
- Return `{ chunks, warnings }`
