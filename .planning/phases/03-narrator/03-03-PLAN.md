**File 2: `packages/cli/src/commands/narrate.ts`**

Create the narrate CLI command following the established pattern from scan.ts.

```typescript
import { readFile, writeFile, mkdir } from 'node:fs/promises'
import { resolve, dirname } from 'node:path'
import chalk from 'chalk'
import ora from 'ora'
import { narrate, format, createProvider, TimelineSchema, FormatTypeSchema } from '@buildstory/core'
import type { FormatType, NarrateOptions, LLMProvider } from '@buildstory/core'
import { loadConfig } from '../config.js'
```

`narrateCommand(timelinePath: string, opts)` where opts: `{ config?: string; format?: string; provider: string; style: string; output: string }`:

1. Load config via `loadConfig()` (same pattern as scan.ts)
2. Read `timelinePath` from disk: `readFile(resolve(timelinePath), 'utf-8')`
3. Parse and validate: `TimelineSchema.parse(JSON.parse(raw))`
4. Determine provider: `config.provider ?? opts.provider` (cast to `'anthropic' | 'openai'`)
5. Determine API key based on provider:
   - anthropic: `process.env['ANTHROPIC_API_KEY'] ?? ''`
   - openai: `process.env['OPENAI_API_KEY'] ?? ''`
6. Build NarrateOptions: `{ provider, style: config.style ?? opts.style, apiKey, maxInputTokens: config.narrate?.maxInputTokens }`
7. Create LLMProvider via `createProvider(narrateOpts)` — create once, reuse for narrate and format
8. Spinner: "Narrating timeline..."
9. Call `narrate(timeline, narrateOpts)` — returns StoryArc
10. Spinner succeed: "Narration complete — {arc.beats.length} beats extracted"
11. Write StoryArc JSON: `writeFile(resolve(opts.output, 'story-arc.json'), JSON.stringify(arc, null, 2))`
12. Determine format types per D-05:
    - If `opts.format` specified: validate against FormatTypeSchema, use `[opts.format as FormatType]`
    - If not specified: use all 4: `['outline', 'thread', 'blog', 'video-script']`
13. For each format type:
    - Spinner: "Generating {formatType}..."
    - Call `format(arc, ft, llmProvider)` — pass the LLMProvider instance directly
    - Write to `resolve(opts.output, `${ft}.md`)`
    - Spinner succeed
14. Final summary: log the list of files written

Ensure `mkdir(opts.output, { recursive: true })` before writing any files.
